
# BrightTV Viewership Analytics Case Study

Overview

BrightTV's CEO aims to increase the company's subscription base during the current financial year. This project analyzes subscriber and viewership data to uncover user behavior patterns, consumption trends, and opportunities for growth. The analysis focuses on generating actionable insights that can support the Customer Value Management (CVM) team in improving engagement, increasing content consumption, and attracting new subscribers.

Case Study Objective

The primary objectives of this analysis were to:

* Analyze user and viewership trends across the BrightTV platform.
* Identify factors that influence content consumption.
* Recommend content strategies to improve engagement during periods of low consumption.
* Propose initiatives that could contribute to subscriber growth and retention.
* Provide data-driven insights to support business decision-making.

Dataset Notes

* The dataset contains subscriber profile information and viewer transaction data.
* Each record in the transaction dataset represents a single viewing session.
* All timestamps were originally provided in UTC and were converted to South African Standard Time (SAST, UTC+2) for analysis.
* Data preparation included validating data quality, handling missing values where necessary, and creating time-based features to support trend analysis.
* Additional metrics and insights were derived to better understand viewing behavior and consumption patterns.

Tools & Technologies

* SQL – Data cleaning, transformation, and analysis.
* Microsoft Excel – Data exploration and validation.
* Power BI – Dashboard creation and data visualization.
